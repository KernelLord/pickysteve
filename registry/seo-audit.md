# Technical SEO Audit

Use when a site needs a technical SEO pass — indexing problems, ranking stagnation,
or a pre-launch checklist. Typical triggers: our pages aren't showing up in search,
blog posts aren't getting indexed by Google, the site isn't ranking, crawl problems.

## Capabilities
- Crawlability: `robots.txt`, `noindex` misuse, canonical tags, redirect chains.
- On-page metadata: titles, descriptions, heading hierarchy, Open Graph tags.
- Structured data: schema.org / JSON-LD validation for rich results.
- Sitemaps: dynamic XML sitemap correctness and submission.
- Core Web Vitals (LCP/CLS/INP) impact on rankings and how to fix the worst offenders.
- Internal linking and information architecture for crawl-budget efficiency.

## Notes
Distinguishes technical SEO from content/AEO. Produces a prioritized fix list.
