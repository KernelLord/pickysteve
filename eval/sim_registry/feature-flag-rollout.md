# Feature Flag Percentage Rollout

Use to roll a change out to a PERCENTAGE of users or TRAFFIC — e.g. serve it to 5% or 10% of requests/traffic first — and roll back instantly via a flag if errors spike. This is a canary / percentage rollout. NOT a full environment swap (blue-green-deploy).
