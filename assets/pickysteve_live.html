Plugs into every agent you already use

MCP stdio native

Claude Code, Cursor, Windsurf, Cline, Codex, Gemini CLI, Goose… one line: claude mcp add pickysteve -- PYBIN -m pickysteve.connectors.mcp_server The agent calls pick_context(); this page mirrors each call live.

OpenAI-compatible proxy

Aider, Hermes, anything with a custom base URL → http://localhost:8077/v1. Picking happens transparently on every request; the visualizer taps the same trace log.

REST / CLI / import

Universal fallback: POST /pick, python -m pickysteve "task", or Engine().pick(). Every call appends to logs/runs.jsonl — the live feed this page replays.

One-command install

python -m pickysteve.connectors.install --all detects your agents and wires them (with backups + clean uninstall). Then open PickySteve Live in a browser tab and watch every routing decision as it happens.
